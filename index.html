University

Low-cost Wearable Movement Data Collection Device (2023 - 2024)
University dissertation motivated by the lack of affordable movement data collection devices in movement disorder research. The project consists of firmware for an Arduino device used to collect the data and an Android application for wirelessly configuring the device and annotating collected data using an activity recognition ML model. The source code and documentation can be found on GitHub.
